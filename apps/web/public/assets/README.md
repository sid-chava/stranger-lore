# Assets Directory

Place the following image files in this directory:

## Required Images

1. **stranger-lore-logo.png** - Primary Stranger Lore wordmark used in the site header
   - Provide at least a 2x resolution PNG with transparency
   - Keep colors aligned with the neon gradient palette used on the site

2. **background.png** - Background image with starry night sky and pixelated grey clouds
   - Should be full screen size (1920x1080 or larger recommended)
   - Format: PNG with transparency if needed

3. **demogorgon.png** - Pixelated red and white Demogorgon character
   - Size: Approximately 120x120px or similar
   - Format: PNG with transparency

4. **social-x.png** - X/Twitter social media icon
   - Size: 24x24px
   - Format: PNG with transparency

5. **social-instagram.png** - Instagram social media icon
   - Size: 24x24px
   - Format: PNG with transparency

6. **social-tiktok.png** - TikTok social media icon (or musical note icon)
   - Size: 24x24px
   - Format: PNG with transparency

## Image Requirements

- All images should be pixelated/retro style to match the 8-bit aesthetic
- Use PNG format for transparency support
- Ensure images are optimized for web (consider using tools like ImageOptim or similar)
